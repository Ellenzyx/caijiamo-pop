### Overview

`CAIJIAMO POP` is a mobile‑first web puzzle game. Open the link and play instantly.

- **Core gameplay**: On a 7×7 grid of dishes, tap adjacent tiles of the same type; selecting 3 cooks one CAIJIAMO.
- **Timer**: Each round lasts 60 seconds. The time bar turns red and shakes in the last 10 seconds.
- **Celebration**: When the round ends, a pixel‑style sparkle/star animation plays with a “bulingbuling” victory sound.
- **MY CAIJIAMO**: All CAIJIAMO cooked in the current round stay visible; if the area gets full, the icons auto‑shrink to fit until the game ends.
- **MENU**:
  - All dishes are shown. After each finished round, **2 new dishes are unlocked**.
  - Unlocked dishes are shown in color and can be opened for **Chinese/English** recipes.
  - Locked dishes are grayed out and cannot open recipes (you can still favorite them).
- **CALENDAR**:
  - MONTH view: tap a day to toggle a small `caijiamo1` icon (“Did you have a caijiamo today?”).
  - YEAR view: shows a per‑month frequency bar chart, normalized by days in each month; supports switching years.
- **UI feel**: Buttons use a consistent pixel click sound; the game supports mobile touch.
