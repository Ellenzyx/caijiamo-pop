## CAIJIAMO POP / 菜夹馍消消乐

**在线试玩**：`https://caijiamo-pop.vercel.app`

[English](README.md) | [中文](README.zh.md)

### 介绍

`CAIJIAMO POP` 是一个为手机优先设计的网页小游戏：点开链接即可玩，无需下载。

- **玩法**：在 7×7 的菜品棋盘上，点击相邻的同类菜品，凑满 3 个即可做出一份菜夹馍。
- **计时**：每局 60 秒；最后 10 秒时间条会变红并抖动提醒。
- **庆祝效果**：回合结束进入 `CAIJIAMO READY!` 时，会播放像素风闪光/星星动画，并伴随 bulingbuling 胜利音效。
- **MY CAIJIAMO**：本局做出的菜夹馍会一直显示在下方区域；如果数量太多占满区域，会自动缩小图标，直到游戏结束。
- **菜单（MENU）**：
  - 默认展示全部菜品；每完成一局会**解锁 2 个新菜**。
  - 已解锁菜品为彩色，可点进查看**中/英文**做法。
  - 未解锁菜品为灰色，不能查看做法（但可以点小心心收藏）。
- **日历（CALENDAR）**：
  - MONTH：点某一天可以标记“今天吃过菜夹馍”，显示为小 `caijiamo1` 图标。
  - YEAR：用条形图展示每月吃菜夹馍的频率（按当月天数归一化），并支持切换年份。
- **操作手感**：按钮统一像素点击音效，支持手机触摸操作。
